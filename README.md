# Skienmarka - Gruver & Severdigheter

Dette prosjektet er et interaktivt kart over Skiensmarka («Gruvekart/Vox Portal») med oversikt over nedlagte gruver, kulturminner, mineralressurser, reguleringsplaner, samt geologiske data (radon, løsmasser, berggrunn, kvikkleire) og eiendomsinformasjon. Det benytter data og kartlag fra blant annet NGU, Kartverket og Riksantikvaren for å tilby et samlet, brukervennlig kart.

Hovedmålet med prosjektet er å bevare og visualisere lokalhistorisk data og gjøre dette tilgjengelig for allmennheten. Posisjonsdata og detaljer om de historiske stedene er lagret i systemet (`data.json`).

## Åpne data: Fritt frem for gjenbruk!

**All data og alle koordinater/punkter som er samlet inn i dette prosjektet kan fritt benyttes av hvem som helst i andre prosjekter.** 

Dersom webhotellet eller de opprinnelige tjenestene noensinne skulle forsvinne, fungerer denne GitHub-repositoryen som et offentlig, varig arkiv. Ønsker du å eksportere dataene, bygge en egen applikasjon eller utføre egne analyser, står du fritt til å bruke innholdet og punktene til ditt eget formål.

Gjenbruk er oppmuntret for å sikre at historien og kunnskapen om stedene lever videre i mange år fremover.
